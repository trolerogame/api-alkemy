
### INSTALL PACKEGAS

```bash
    npm i
```

### ENVIRONMENT VARIABLES

```env
    # mysql server
    PASSWORD=my_password
    HOST=localhost
    USERNAME=root
    PORT=5050  # port server
    SECRET=secretkey  # secret key for jwt
    DB=my_db
    PORTDB=port database
```


### Start dev server

```bash
    npm run dev
```
